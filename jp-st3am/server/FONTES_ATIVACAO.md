# Fontes de Instruções de Ativação

Pesquisa realizada para instruções de ativação de cada jogo (como ativar após baixar).

## Fontes consultadas

| Fonte | Conteúdo |
|-------|----------|
| **CrackRelease** | Lista de jogos Denuvo cracked/bypassed, grupos (EMPRESS, MKDEV, etc.) |
| **Anadius** | Origin/EA emulator, fluxo Denuvo ticket para jogos EA |
| **GameCentr** | Guia offline Denuvo, Steam offline |
| **Goldberg Emulator** | Steam emulator para jogos sem Denuvo |
| **CrackWatch** | Status de crack por jogo |
| **Gamebounty / Gamedrive** | Páginas por jogo com instruções |

## Tipos de ativação

- **denuvo_ticket**: Execute jogo → gera .txt → envia aqui → recebe token → coloca em config
- **bypass**: Baixa crack/bypass → extrai na pasta → executa
- **ea_bypass**: EA App + VPN (Showcase) + Showcase Magic/Bypass.bat
- **ea_anadius**: Anadius emulator + ticket + token
- **steam**: Launcher ou Goldberg emulator
